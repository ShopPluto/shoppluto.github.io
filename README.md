# 📖 About Pluto

Pluto is a modern e-commerce platform focused on delivering a premium shopping experience through clean design, high performance, and scalable architecture.

The project is being developed from the ground up using professional software engineering practices. The goal is not only to build a functional shopping platform but also to create a maintainable, production-ready codebase.
